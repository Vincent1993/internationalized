# Internationalized 国际化格式化套件

欢迎来到文档站点！这里汇总了 `internationalized` Monorepo 中的数字与日期时间格式化能力，帮助你在前端、服务端与脚本环境中统一调用。

> 文档由 [Rspress](https://rspress.dev/) 驱动，使用 `pnpm` 管理依赖，并通过 `corepack` 保持团队环境一致。

- 阅读[快速上手](./guide/getting-started.mdx)，了解如何在项目中安装 `@internationalized/number-format` 与 `@internationalized/date-format`。
- 查看[格式化能力](./guide/formatting.mdx)，通过示例快速了解常见场景。
- 深入[数字格式化包概览](./guide/number-format.mdx)与[日期格式化包概览](./guide/date-format.mdx)，查阅模块地图、目录结构与能力矩阵。
- 使用[模板字符串](./guide/templates.mdx)高效描述复杂的数字展示逻辑。
- 深入[插件系统](./reference/plugins.mdx)与[架构总览](./reference/architecture.mdx)，理解扩展点与执行顺序。
- 在[FP 函数参考](./reference/fp.mdx)中检索数字与日期的纯函数 API，配合[解析说明](./reference/parser.mdx)处理输入输出。
- 参考[集成与自动化](./guide/tooling.mdx)，建立 lint、测试与覆盖率的统一流水线。

如有问题或建议，欢迎访问 [GitHub Issues](https://github.com/Vincent1993/internationalized/issues) 反馈。
